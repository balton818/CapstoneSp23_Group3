Edit Ingredient Quantity Use Case Scenario (Mobile)

Context - User has successfully logged in and clicked the My Pantry

PRIMARY FLOW
1 - Current Pantry contents is displayed to user.
2 - User selects ingredient 
3 - User Clicks plus button
4 - Ingredient amount is increased and pantry is updated

ALTERNATIVE FLOW
A3 - User Clicks Minus Button
A4 - Ingredient amount is decreased and pantry is updated
